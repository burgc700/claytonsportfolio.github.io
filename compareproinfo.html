About the project

A full stack web application that allows users to compare computer parts like CPU, GPU, SSD, and RAM between the NewEgg and MicroCenter websites. Created to make it easier for users to do research on computer components by having it all in one window, to eliminate the use of using a 3rd party tool like excel to do research. Also, made easy for users to find the products they want to keep researching by allowing users to like the products and have those products in their own page. Each type of product can be selected to see all the products available of that type. The user can then sort those products by selecting a minimum price and minimum rating.

After the user finds a product they want to investigate the product page will show all the details of the product including features and where the price break down is between the two sites. Each price comparison has a link that will take the user to the product page on the site so they can make a purchase if they want.

The user can add comments to the product as well. This is to do their research on products they are deciding to buy when picking between more than one category. Each comment allows the user to select a field that are either prepopulated or the user can add their own. Each field is grouped together so users can find like comments.

Users can also compare products. This works based off the comments for a product. Thinking like users will only want to compare products they are actively researching. So, the user has to have a comment(s) in at least two product pages in order to compare products.

After the user selects a product to compare they will see the current product and the selected product that make up the comparison. It shows all prices and ratings from the sites as well as the comments sorted by the field just as in the previous page. Not pictured but also present the product image and a button that takes the user to that products product page.
